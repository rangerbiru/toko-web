FAQ

Masih kebingungan ya? Lihat disini ya

Apa Itu Website ?

Website adalah kumpulan halaman dalam suatu domain yang memuat tentang berbagai informasi agar dapat dibaca dan dilihat oleh pengguna internet melalui sebuah mesin pencari. Informasi yang dapat dimuat dalam sebuah website umumnya berisi mengenai konten gambar, ilustrasi, video, dan teks untuk berbagai macam kepentingan.

Mengapa Harus Memiliki Website ?

Karena website sangatlah berguna untuk memperkenalkan diri atau usaha, agar dapat dengan mudah ditemukan serta dikunjungi oleh pengguna internet yang ada di seluruh dunia. Tentunya akan sangat bermanfaat bagi UMKM terkhususnya di Indonesia, agar mempermudah kita dalam menjangkau target pemasaran ke segala penjuru di Indonesia dan bahkan di seluruh dunia.

Apa Itu Website Responsive ?

Website Responsive merupakan sebuah keadaan pada tampilan halaman website yang akan melakukan sebuah penyesuaian, sesuai dengan perangkat yang anda/pengunjung gunakan saat mengunjungi website tersebut.

Apa Itu Hosting Dan Domain ?

Hosting merupakan sebuah ruang penyimpanan data yang sangat besar atau bisa disebut juga sebagai file pada komputer yang diupload ke dalam sebuah server.
Domain merupakan komponen yang berfungsi memberi pengarahan untuk menuju ke alamat server hosting atau IP address, pengelolaan nama domain sendiri berada di bawah pengaturan instansi regulasi domain global atau ICANN.

Apakah Saya Akan Mendapatkan Akses Hosting Dan Domain ?

Akses untuk Hosting dan Domain hanya tersedia pada pembelian paket SEWAJARNYA dan SEBAGUSNYA. Namun jika anda memilih paket SEIKHLASNYA, anda juga bisa mendapatkan akses Hosting dan Domain dengan hanya membayar biaya tambahan sebesar Rp. 50.000

Apakah Saya Boleh Meminta Revisi ?

Yaa tentu, kami memberikan revisi pada pembuatan website sesuai dengan jumlah revisi yang kami berikan pada setiap paket pembelian. Untuk lebih jelasnya, dapat dilihat pada halaman Price List kami.

Apakah Saya Dapat Menggunakan Template Website Yang Saya Miliki ?

Yaa tentu, silahkan hubungi kami apabila anda memiliki sebuah template website yang ingin anda gunakan untuk website anda.

Bagaimana Cara Saya Mengelola Website Saya Nantinya ?

Kami akan memberikan petunjuk pada setiap source code yang akan kami berikan untuk anda, guna untuk mempermudah anda dalam mengelola website nantinya. Dan kami juga menyediakan layanan jasa maintenance untuk website anda, dengan cukup membayar seikhlasnya di tiap dilakukannya perubahan pada website.
